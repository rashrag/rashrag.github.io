Automated Video Surveillance

A research project to automate real-time detection of suspicious activity in surveillance footage using cloud infrastructure to support real time processing of videos. We implemented the support vector machine algorithm(using four features) in a cloud computing environment with a map reduce technique using Matlab’s Parallel Computing Toolbox that yielded a 70% accuracy in object detection. We introduced a suspicion value for every new object in the frame and re-calculated the same on the fly depending on the context of its appearance. During the first four months we performed all computations on a single computer with multiple processors. To improve real time detection, we shifted from a single node to a four node approach, with support from Matlab Distributed Computing Server. We saw an increase in speed of detection of objects and suspicious value re-calculation by 60% and 65% respectively.

Congruent Networks

Congruent Networks is a novel concept and model for utilizing electronic social networks in a context. As part of this project, I analysed how the two different psychological groups – introverts and extroverts behave in social network environments. In consultation with experts from National Institute of Mental Health and Neurosciences, Bangalore, India, I created a questionnaire to categorise a person into either group and monitored their behaviour in social media (Facebook and Twitter) using simple API invoking code. Our dataset included 100 people and conclusions on the time spent, posts and comments, ease of usage and addiction among the groups were made. The results revealed that extroverts spent more time, posted more and found social media easier to use. However, introverts were found to be more addicted to social media. This research was discontinued due to lack of funds to scale the project.
