Social Data Aggregator and Locator of Knowledge

As part of my final year project, in a team of 3, I worked on a social data Aggregator System. This sytem involved getting social network information from across different social media sites. Both direct and derived information can be viwed. The subject can submit a subset of his links for any event. A user can create an event and shortlist subjects for the same. There are numerous use cases to this project. It can be used for job screening, hackathon shortlisting, targetted advertisements and bheavior detection( to find cases of bullying or life threathening situations). We used GIT,Linkedin and Facebook as proof of concepts. A query system was also implemented for fast access of data

Dialogue System

As part of my Natural Language processing course, in a team of four,we created a dialogue system between a potential phone user and the simulated e-commerce site. The entire class taking the course generated the custom database on Mobile systems. We implemented the Maxent Model with the Veterbbi Algorithm to tag entities in the questions. Once the words are agged, the context of the question was understood using entity-relationship mapping. Once the question was understood, the answer was fecthed and through natural language generation displayed to the user. This system was developed over the span of 3 days as the final examination project.

TestCo

As part of my Software Engineering course, in a team of fourten,we developed an online testing system. As part of the system, we developed a system, where students could take a test. The system contained a question pool which teachers contribute to. The level of difficulty of the examination was set by the admin. The system has capabilites to increase the level of difficulty depending on the previous answers given by the student. The system provided for both multiple choice questions as well subjective ones. We also provided features like drawing boards and calculators. The project was developed using an iterative approach of software engineering. We used HTML5, PHP and MongoDB to develop the same.
