Set 4

Set 4 is a multiplayer card game. At the start of the game, each of the four players are given random set of 4 cards. The deck contains four cards each of the number 1,2,3,4. The aim of the game is to keep passing cards so as to collect four cards of the same number. There is a java standalone app as well as an adroid application. The game can be played with other players or with the computer.

Titanic

Titanic is an HTML5 game. Inspired from Flappy Bird, the aim of the game is to control and navigate a ship around ice bergs. The more icebergs crossed, the speed of the ship increases. There are different speed modes with which the game can be played. The scores can be logged online and compared with the score of your friends. The game can be played on web browsers and phones that support HTML5
