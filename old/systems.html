MapGraph: Parallel Graph Database

As part of my big data elective course, I worked on Map Graph, an API for parallel graph analytics on GPU, before it was commercialized into Blazegraph products.We provided feedback on compatibility between CUDA versions and the API versions. As part of the proof of concept, we implemented new algorithms such as Graph 2- colorability and Cycle detection. These and other pre-existing algorithms were compared against Goffish and Giraph databases which are distributed graph database frameworks. We also setup a GPU Cluster using free and open source projects.

Online Scrabble


As part of my Unix System Programming course, in a team of five,we designed an online scrabble system. This system would allow a room with upto four players and multiple rooms to run at a single instance. Our design had to ensure concurrency of the games and message passing between the rooms. Players waiting could wait for the next free room or create a new room. Players in an existing room could chose to start a new game there, or exit. Points in the game had to be accounted for.
